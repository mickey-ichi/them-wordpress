Upload Theme↑ Back to Top

After you complete Installation of WordPress, Please follow below Steps to install theme:

You have two ways to install a theme.



1. Upload Theme from Admin panel

Step 1: Login to your WordPress admin area.
Step 2: Go to Dashboard > Appearance > Themes.
also see below screenshot.



Step 3: Now, Click on Upload Theme button at top and upload megashop.zip also see below screenshot.



also see below screenshot.



2. Upload Theme from FTP

Do not upload the Entire Zip File you get from TemplateTrip. Please follow following Steps to upload theme via FTP:

Step 1: Using your FTP program, upload the non-zipped theme folder into the "/wp-content/themes/" folder on your server.
Step 2: Go to Dashboard > Appearance > Themes and find Megashop theme from list
Step 3: Click on Activate to active Megashop theme

WooCommerce - excelling eCommerce Plugin ↑ Back to Top

Go to Admin Panel > Woocommerce > Settings > Products to change the Products settings.

Step 1: Click on Products tab
Step 2: Click on display tab
Step 3: Select "Shop" as Product Archive / Shop Page
Step 4: Change dimentions of Product Image Sizes
Step 5: Save Changes

also see below screenshot.



WooCommerce Account Settings.

Step 1: Click on Accounts tab
Step 2: Select "My Account" as My Account Page
Step 3: CHECK Enable registration on the "Checkout" page
Step 4: Save Changes



WooCommerce Checkout Settings.

Step 1: Click on Checkout tab
Step 2: Select "Cart Page" as Cart Page
Step 3: Select "Checkout Page" as Checkout Page
Step 4: Save Changes
